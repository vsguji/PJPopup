#  PJPopup

![效果图](https://github.com/vsguji/PJPopup/blob/master/Popup/ScreenShot_20190405182437.png)

Animation:

罗列几种弹出内容视图效果

Fade  淡入

Grow    渐变

Shrink  缩放

Slide   平滑

Bounce 弹性


Orientation:

支持根据屏幕旋转
